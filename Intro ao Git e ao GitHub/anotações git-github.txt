PRIMEIROS COMANDOS COM O GIT
*ls > listar
*ls -a > flag que mostra arquivos ocultos dentro da pasta
*cd> entrar na pasta
*Control+L > limpar tudo
*mkdir > criar outra pasta
*git init > inicializar o git
*git add nome do arquivo > move arquivo para staged
*git add *> pega tudo que foi modificado no diretorio de trabalho e 	direciona para o staged
*git commit > envelopa esse arquivo e coloca ele em commit
*git status > ajuda a monitorar os status dos arquivos
